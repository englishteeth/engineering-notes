# Templates

Starting points for engineering practices that benefit from structure.

These aren't rigid forms to fill in — they're prompts for the kind of thinking that's easy to skip under pressure.

| Title | Description |
|---|---|
| [Architecture Decision Record](adr-template.adoc) | A lightweight template for recording architectural decisions and the reasoning behind them |
| [Incident Post-Mortem](incident-post-mortem.adoc) | A structured template for learning from production incidents without assigning blame |
